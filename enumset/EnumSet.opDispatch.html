• enumset

    Template EnumSet.opDispatch

    Access the value at the index specified by the name of an enum member.

    The value is returned by reference, so it can used for assignment. set.name is just syntactic sugar for set[SomeEnum.name].

    Arguments

    template opDispatch(string s);

    Functions

    Function name Description
    opDispatch

    Example

    EnumSet!(Element, int) elements;
elements.water = 5;
assert(elements.water == 5);
